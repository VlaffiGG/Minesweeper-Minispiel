Ausführung des Spiels:

-Um das Spiel auszuführen, folge diesen Schritten:
 ->Öffne die Datei index.html in deinem bevorzugten Webbrowser.
 ->Das Vinesweeper-Spiel sollte nun geladen und spielbereit sein.
 ->Klicke auf die Zeilen, um sie zu öffnen und das Spiel zu spielen.


Spielanleitung:

-Das Ziel des Spiels ist es, alle Zeilen aufzudecken, ohne auf eine "dornige Ranke (Vine)" zu klicken. 
 Wenn du auf eine Zeile klickst, wird entweder eine Zahl angezeigt, die dir sagt, wie viele Vines sich in den angrenzenden 
 Zeilen befinden, oder eine leere Fläche, die sich automatisch ausbreitet und alle leeren Zeilen umfasst.

-Klicke mit der rechten Maustaste auf eine Zeile, um es mit einer Markierung zu versehen, falls du glaubst, dass sich dort 
 eine Vine befindet. Ein weiterer Rechtsklick entfernt die Markierung wieder.

-Das Spiel endet, wenn du auf eine Vine klickst oder alle Zeilen ohne Vines aufgedeckt sind.

-Viel Spaß! :D
